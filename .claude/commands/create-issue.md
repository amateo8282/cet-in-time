# GitHub 이슈 생성 커맨드

입력된 내용을 분석하여 GitHub 이슈를 생성합니다.

## 입력

$ARGUMENTS

## 수행 작업

### 1. 입력 분석

입력된 "$ARGUMENTS" 내용을 분석하여 다음을 추출하세요:
- **이슈 제목**: 핵심 내용을 간결하게 요약 (한 줄)
- **이슈 본문**: 상세 설명, 배경, 요구사항 등

### 2. 이슈 구조화

이슈 본문은 다음 형식으로 구조화하세요:

```markdown
## 설명
[입력 내용의 상세 설명]

## 배경
[왜 이 이슈가 필요한지]

## 완료 조건
- [ ] [구체적인 완료 조건]
```

### 3. 이슈 생성

`gh issue create` 명령어를 사용하여 이슈를 생성하세요:

```bash
gh issue create --title "[이슈 제목]" --body "[구조화된 본문]"
```

### 4. 결과 출력

생성된 이슈의 URL을 사용자에게 알려주세요.

---

## 실행

위 지침에 따라 "$ARGUMENTS"를 분석하고 GitHub 이슈를 생성하세요.
